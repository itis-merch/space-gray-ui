# **Используемые нами цвета из Tailwind CSS**

## **Stone**
  * #fafaf9 (50) - основной белый цвет
  * #0c0a09 (950) - основной темный цвет (для главного текста)
  * #292524 (800) - цвет для побочного текста

## **Yellow**
  * #fde047 (300) - основной желтый цвет
  * #facc15 (400) - желтый цвет для элементов с `hover` свойством
